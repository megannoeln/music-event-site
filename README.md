This was a collaborative project for my Web Applications 2 course.
The project focused on learning how to implement MVC and learning CSS.

Its a web application that allows users to create/update profiles, create events, like/rate existing events, and upload profile pictures.

Navigate to web2->Content->site.css to see CSS files
Navigate to web2->Controllers to see all controller pages (c#)
Navigate to web2->Models to see all model pages (c#)
Navigate to web2->Views to see all view pages (html/cshtml)
